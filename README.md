\# Hugging Face Text Classification Baseline



这是我的第一个 Hugging Face NLP 小项目。



\## 项目目标



从手写 PyTorch LSTM 文本分类，过渡到使用 Hugging Face 预训练模型完成文本分类任务。



\## 当前基础



我已经学习并练习过：



\- PyTorch 基础训练流程

\- MLP 图像分类

\- CNN 图像分类

\- RNN 名字分类

\- LSTM 文本分类

\- ResNet 迁移学习



\## 项目计划



1\. 准备文本分类数据集

2\. 使用 Hugging Face tokenizer 处理文本

3\. 加载预训练模型

4\. 微调模型完成文本分类

5\. 记录实验结果

6\. 整理成可展示项目



\## 与已有学习内容的关系



之前的 `PyTorch\_Learn` 主要练习手写 PyTorch 模型。



本项目会进一步学习：



\- Hugging Face Transformers

\- 预训练语言模型

\- 文本分类 fine-tuning

\- 实验记录和项目组织
